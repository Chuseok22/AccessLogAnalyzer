# 출입기록분석기 v1.0.0

## 주요 기능

- 엑셀 파일 기반 출입 기록 분석 도구
- 의심스러운 출/퇴근 패턴 자동 감지
- 날짜 범위 필터링 기능
- 열 이름 자동 감지 (A, B, I열 기본값 + 문자열 기반 열 인식)
- 분석 결과 엑셀 내보내기

## 다운로드 및 실행 방법

1. 첨부된 `출입기록분석기.exe` 파일을 다운로드
2. 다운로드한 파일을 더블클릭하여 실행

※ Python 설치 없이 Windows 환경에서 바로 실행 가능합니다.
