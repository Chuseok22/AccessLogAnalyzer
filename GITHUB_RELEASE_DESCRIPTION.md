# 초과근무 분석 도구 v2.0.0

## 초과근무 분석 도구 기능

- 경비 기록과 초과근무 기록을 함께 분석
- 경비 작동 중 초과근무 기록 감지 기능
- 평일 9:00-18:00 외 초과근무 자동 계산
- 새벽 4시 기준 근무일 구분
- 15분 이상 의심 사례 자동 필터링

## 다운로드 및 실행 방법

1. 첨부된 `overtime_analyzer.exe` 파일을 다운로드
2. 다운로드한 파일을 더블클릭하여 실행

※ 두 프로그램 모두 Python 설치 없이 Windows 환경에서 바로 실행 가능합니다.
